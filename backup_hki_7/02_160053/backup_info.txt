CARA RESTORE:
1. Copy folder source_code ke lokasi yang diinginkan
2. Restore database dari file database_backup.sql
3. Copy .env.backup menjadi .env dan sesuaikan konfigurasi
4. Jalankan: composer install
5. Jalankan: npm install
6. Jalankan: php artisan key:generate
7. Jalankan: php artisan migrate (jika diperlukan)
8. Jalankan: php artisan storage:link
